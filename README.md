# Atividade-10-04
Atividade de Análise de Dados sobre Modelagem Preditiva

O objetivo desta etapa é desenvolver modelos de Machine Learning capazes de prever o custo de internações hospitalares a partir dos dados tratados nas etapas anteriores. A proposta é transformar os insights obtidos na análise exploratória em capacidade preditiva, aplicando técnicas de pré-processamento, treinamento e validação de modelos.

Além disso, busca-se comparar diferentes algoritmos para identificar qual apresenta melhor desempenho, garantindo que a escolha final seja baseada em métricas quantitativas e alinhada ao comportamento dos dados analisados.
